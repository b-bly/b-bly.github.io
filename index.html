About

I've done development on all parts of the stack--front and back end. On the front end I have mainly used React/Redux and Angular.js to make dynamic sites with reusable code. I use media queries to make sites mobile-responsive.

On the back end I've used node.js/express and java spring to code efficient apis. I've worked with postgreSQL, mySQL and mongoDB to manage data.
